# **App Name**: TrustEarn

## Core Features:

- Task Marketplace: A curated list of companies requiring reviews, allowing users to browse and select tasks to perform.
- One-Tap Link Out: Seamlessly navigate to the specific company's Trustpilot page via direct integration links.
- Review Generation Tool: An AI-powered tool that helps users structure their honest experiences into well-written reviews based on company attributes.
- Submission Portal: Interface for users to submit screenshots or review IDs as proof of their task completion.
- Reward Wallet: A dashboard to track pending rewards, earned cash, and withdrawal history.
- Payout Management: Integration with payment providers to facilitate cash rewards to user bank accounts.

## Style Guidelines:

- Primary color: Deep Royal Indigo (#4F46E5) evoking professional trust and institutional reliability.
- Background color: Sophisticated Obsidian (#0B0E14), a heavily desaturated dark base that enhances content clarity and modern appeal.
- Accent color: Bright Sky Blue (#38BDF8) providing a sharp, high-contrast point of interest for interactive elements and call-to-actions.
- A font pairing of 'Poppins' (Sans-serif) for high-impact headlines and 'Inter' (Sans-serif) for readable body text, creating a modern, objective look.
- Thin-line, minimalist monochrome icons that reflect financial growth and verified social proof.
- A spacious, card-based dashboard layout with ample padding to maintain an organized, premium fintech atmosphere.
- Smooth, subtle CSS transitions for hover states and sleek, incremental number animations for the user balance display.
